Quick Start

1

Boot from Void Linux Live ISO

Download the Void Linux live ISO and boot from it.

2

Run the installer

Connect to the internet and run:

xbps-install -Syu xbps curl && curl -sL https://mggpie.github.io/void-installer/bootstrap.sh | sh
3

Follow the prompts

The installer will ask for:

  • LUKS encryption passphrase
  • Root password
  • User password

  • All data on the target disk will be erased!

Configuration

Edit config.example.sh before installation to customize:

  • TARGET_DISK - Target disk (auto-detected by default)
  • HOSTNAME - System hostname
  • USERNAME - Primary user account
  • TIMEZONE - System timezone
  • FILESYSTEM - ext4, btrfs, or xfs
  • SWAP_SIZE - Swap file size or "auto"

What Gets Installed

  • Base: Void Linux (glibc) with runit init
  • Bootloader: GRUB with LUKS support
  • Networking: dhcpcd or NetworkManager
  • Utilities: curl, wget, git, vim, sudo

This is a minimal installation. Add your packages and dotfiles after first boot.
